# 2.4.2 ページタイトル

## 達成基準
ウェブページには、主題又は目的を説明したタイトルがある。
## 達成方法
`title`要素を用いて、ウェブページに説明的なページタイトルを提供する
## 実装ガイドライン
- `title`要素に、各ページのタイトルを設定する。
```HTML
<html>   
   <head>     
      <title>Title</title>     
   </head>   
   <body>     
   </body> 
</html>  
```
