# 3.2.1 フォーカス時

## 達成基準
いずれのコンポーネントも、フォーカスを受け取ったときに コンテキストの変化を引き起こさない。
## 達成方法
コンテキストの変化に対するトリガーとして、`focus`ではなく、`activate`を使用する

## 実装ガイドライン
- フォーカスを受け取ったときに、ユーザーが意図しない変更が起こらないようにする。（フォームが自動送信される、次のページが開く等）
