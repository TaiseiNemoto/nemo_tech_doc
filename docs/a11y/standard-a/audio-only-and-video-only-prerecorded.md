# 1.2.1 音声のみ及び映像のみ (収録済)

## 達成基準
収録済の音声コンテンツ及び収録済の映像コンテンツは、代替コンテンツを提供する
## 達成方法
### 収録済の音声しか含まないコンテンツの場合
- 音声のみの時間依存メディアに対する代替コンテンツを提供する

### 収録済の映像しか含まないコンテンツの場合
- 映像のみの時間依存メディアに対する代替コンテンツを提供する
- 重要な映像コンテンツを説明する音声を提供する
