# ★2.5.2 ポインタのキャンセル

## 達成基準
シングルポインタを使って操作できる機能は、以下の要件の少なくとも 一つを満たす。
- ダウンイベントがない
- 中止又は元に戻すことができる
- アップイベントで反転
- 必要不可欠
## 達成方法
- ドラッグ＆ドロップ操作をキャンセルできるようにする
- ネイティブコントロールを使用して、アップイベントで機能がトリガーされるようにする
