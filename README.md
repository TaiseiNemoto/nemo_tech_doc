# Getting Started

## Node
`16.0.0`以上を推奨します。

## セットアップ
```
npm install
or
yarn install
```

## 開発サーバーの起動
```
npm run docs:dev
or
yarn docs:dev
```

## ビルド
```
npm run docs:build
or
yarn docs:build
```

## プレビュー
こちらのコマンドを実行すると、`.vitepress/dist`内のファイルからローカルの静的Webサーバーを起動し、`http://localhost:4173`でアクセスすることができます。本番用のビルドが、ローカル環境で正常に見えるかどうか確認することができます。
```
npm run docs:preview
or
yarn docs:preview
```
